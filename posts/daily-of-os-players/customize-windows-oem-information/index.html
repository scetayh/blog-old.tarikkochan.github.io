自定义Windows OEM信息

在 Windows 设置和关于 Windows（ winver.exe ）中，用户可以查看 OEM 信息。这些信息记录了制造商名、型号名等信息，可高度自定义。

本教程指导用户在 Windows 8.x 及以上自定义 OEM 信息。

# Windows 设置

打开注册表编辑器 regedit.exe ，创建项 计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\OEMInformation 。

新建并修改以下各字符串值：

  • ManuFacturer —— 制造商
  • Model —— 型号
  • SupportHours —— 支持小时数
  • SupportPhone —— 电话号码
  • SupportURL —— 网站

# 关于 Windows（ winver.exe ）

打开注册表编辑器 regedit.exe ，创建项 计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion 。

新建并修改以下各字符串值：

  • CurrentBuild ——OS 内部版本
  • DisplayVersion —— 版本
  • RegisteredOwner —— 许可使用本产品的用户
